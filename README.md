# Random Projects

A collection of small browser games, each a single self-contained `index.html`.

**Play them live:** https://meihuankit.github.io/randomProj/

| Project | Description |
|---|---|
| [Guwahati Walk](https://meihuankit.github.io/randomProj/guwahati-walk/) | A walking exploration game set in Guwahati |
| [Sky Drift](https://meihuankit.github.io/randomProj/sky-flyer/) | A night-sky flying game with multiplayer rooms |

## Running locally

No build step — open any project's `index.html` in a browser, or serve the folder:

```sh
python3 -m http.server
```
